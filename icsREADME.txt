###README###

The program contains two functions. 
The main funtion is changeStartDate. It takes a file name and a date as arguments.
The date is the new start date for the first event on the calendar. 
The last line in the program is where the file and date are inputed.
The program must be in the same directory that the file is located in.

changeStartDate will shift all the events in the calendar over according to the new start date.
